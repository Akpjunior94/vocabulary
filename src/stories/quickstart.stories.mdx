# Quickstart

To start using Vocabulary for quick prototyping or development you can use one of the links from CDN providers. Only add one of these lines within your document `head` tag.

## CDN

### UNPKG

To specific version:

```html
<link rel="stylesheet" href="https://unpkg.com/@creativecommons/vocabulary@1.0.0-beta.6/css/vocabulary.css">
```
Or the link that always points to the latest version:

```html
<link rel="stylesheet" href="https://unpkg.com/@creativecommons/vocabulary/css/vocabulary.css">
```

### jsDelivr

To specific version:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@creativecommons/vocabulary@1.0.0-beta.6/css/vocabulary.css">
```
Or the link that always points to the latest version:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@creativecommons/vocabulary/css/vocabulary.css">
```

When using Vocabulary in local development and want to use asset files like SVG icons, make sure to be runing a server for them, like [this](https://www.npmjs.com/package/http-server), and reference the right path. Once ready try this snippet:
